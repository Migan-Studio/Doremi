# mbpr

- the mbpr (migan-bot-public-repositry) project.

## Using

- **this project is use licence at `MIT`**
- **You need node.js 16.6.0 or higher and yarn**

1. Clone the project.
2. run this command

```sh
yarn install
```

3. copy a `.env` the `example.env`
4. insert Your Bot Token of `TOKEN` value
5. run

```sh
# no build
yarn dev

# build then run
yarn build
yarn start
```
